Primeiros passos

Antes que o AutoHotkey possa fazer qualquer coisa, você precisa escrever um script dizendo ao AHK o que fazer.
Para aprender como, leia o tutorial ou pule direto de cabeça.

Basicamente: crie um arquivo de script → adicione algumas teclas de atalho (hotkeys) → execute o arquivo do script.

Você também pode obter ajuda ou encontrar vários scripts úteis em nosso amigável fórum de discussão.

Por que estou vendo isto?

Porque você executou o AutoHotkey sem dizer-lhe qual arquivo de script ele deve executar, e ele também não pôde encontrar um script em nenhum dos locais comuns.
